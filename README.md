# road-accident-analysis-india
# Statistical Analysis of Road Accidents in India

## Overview
This project analyses road accident data from India to identify key factors influencing accident rates and predict future trends using statistical modelling techniques.

The goal is to generate data-driven insights that can support road safety improvements and policy decisions.

---

## Tools & Technologies used
- Minitab  
- Microsoft Excel  
- Statistical Methods:  
  - Chi-Square Test  
  - Linear Regression  
  - ARIMA (Time-Series Forecasting)

---

## Dataset
- Source: Ministry of Road Transport and Highways (India, 2022)
- Data includes:
  - Weather conditions  
  - Road types  
  - Traffic violations  
  - Accident severity  

---

## Steps taken

### 1. Data Cleaning
- Cleaned raw datasets using Excel  
- Removed inconsistencies and handled missing values  
- Structured data for analysis in Minitab  

### 2. Exploratory Data Analysis (EDA)
- Identified trends and distributions across variables  
- Analysed accident patterns by weather, road type, and region  

### 3. Statistical Analysis
- Conducted Chi-Square tests to examine relationships between categorical variables  
- Built linear regression models to analyse trends over time  

### 4. Time-Series Forecasting
- Developed an ARIMA model to predict future accident rates  
- Evaluated model performance using residual analysis  

---

## Key Findings
- Clear weather conditions showed the highest number of accidents (likely due to higher traffic volume)  
- Straight roads accounted for the majority of accidents  
- Certain regions showed significantly higher accident severity  
- Forecasting models indicate a potential increase in accidents over time
